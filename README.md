# JQuery with Sass

В этом репозитории основы JQuery, React и Sass/Scss

## Сборка 

Для сборки JSX
```bash
npm install babel-cli@6 babel-preset-react-app@3
npx babel --watch src --out-dir ./dist --presets react-app/prod
```

Сборка Sass/Scss в терминале
```bash
npm install -g sass
sass ./css/style.scsss ./dist/style.css
```
